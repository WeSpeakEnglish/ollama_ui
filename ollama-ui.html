Temperature
Controls how random or creative the model is.
0.0 — always picks the safest word
0.7 — balanced default
1.0+ — more varied and creative
2.0 — very random, may be incoherent
Low for code/facts · High for creative writing
Context window
How much text the model holds in memory — your messages + its replies combined.
2 k — fast, short chats
4 k — default, good for most tasks
8 k–32 k — long docs or long conversations
Larger values use more RAM on the server.
Streaming
ON — words appear token by token as the model generates. You see the response immediately.
OFF — model finishes first, then shows the full reply at once.
Recommended ON for long responses.
System prompt
A hidden instruction given to the model before the conversation. Sets its role, tone, or rules.
Examples:
· "You are a Linux expert. Be concise."
· "Always reply in Czech."
· "You are a sarcastic assistant."
The user never sees this instruction.
